Sobre a Definição de Front-end

voltar

Define-se pro front-end toda a estrutura visual de um site que refere-se diretamente à interação do usuário com o site, incluindo elementos como layout e design. Essa interação acontece através de uma interface gráfica, desenvolvida principalmente com HTML e CSS, que permite ao usuário navegar e interagir de forma intuitiva e acessível na webpage.

"O front-end é responsável por toda a interação do usuário com a aplicação".

O que é HTML?

voltar

HTML têm por significado Hypertext Markup Language, sendo utilizada como estruturação de páginas web. HTML, portanto, não é uma linguagem de programação, limitando-se à uma linguagem de marcação de hipertexto.

Como se estrutura um HTML?

voltar

Um HTML é formado por tags que apresentam maior ou menor valor semântico, podendo ser de natureza estrutural, tags de texto, formulários, tags de mídia, listas, tabelas, tags de identificação de atributos e de comentários. Importante salientar que existe a possibilidade de se inserir atributos dentro das tags, a fim de conferir determinada característica à mesma.

Principais Tags do HTML 5 e suas Funções

voltar
  1. HEADER: cabeçalho do site.
  2. NAV: menu de navegação.
  3. MAIN: conteúdo principal da página.
  4. FOOTER: rodapé da página.
  5. H1 a H6: títulos, do maior ao menor.
  6. P: texto do parágrafo.
  7. STRONG: negrito.
  8. EM: itálico.
  9. BR: quebra de linha.
  10. IMG: imagem.
  11. A HREF: link.
  12. UL, OL e IL: lista não ordenada, lista ordenada e item da lista.
